Object.entries()

Returns an array of the key/value pairs in an object

Object.fromEntries()

This method creates an object from alist of key/value pairs

Oject.values()

Object.values() is similar to the Object.entries(), out returns a single dimension array of the object values

Object.groupBy()

  • This method groups the elements of an object according to to string values returned from a callback function
  • This method doesn't change the original object
